# Superstore Sales Analysis Dashboard (Power BI)

## 📊 Project Overview
This project analyzes sales performance across customers, products, and regions using Power BI. The goal is to identify key trends, top-performing segments, and areas for improvement.

## 📁 Dataset
The dataset includes order-level data covering customers, products, sales, and geographic regions.

## ⚙️ Key Features
- Interactive dashboard with slicers (Year, Quarter)
- Customer segmentation and distribution analysis
- Product performance analysis (Category & Sub-category)
- Geographic analysis with map visualization
- Drill-through functionality for customer-level insights

## 📌 Key Insights
- Sales declined by 4% in 2016 before recovering in 2017
- The Consumer segment is the largest contributor to overall sales
- Phones and Chairs contribute 30% of total sales
- The West region leads, with strong performance driven by California

## 🛠 Tools Used
- Power BI
- DAX
- Power Query
